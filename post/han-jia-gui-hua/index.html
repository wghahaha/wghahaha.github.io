寒假规划

2023-01-07 3 min read # 一些思考

其实回家已经有了一段时间了，但是今天才考完试。想的就是今天晚上就看一集小迪的课。然后重点对寒假和下学期进行一些规划。

假期

前端时间羊了，后来又说复习期末考试（其实没有复习，都是在玩。。。）就一直没有搞安全😭😭😭😭。不过还好，还是基本上完成了这学期的给自己规定的至少学完top10和掌握CTF web方面的一些考点。（感觉反序列化和XSS的绕过掌握的不太好，后续也需要对这方面进行加强）。
后面的话想还是想继续看小迪的课，和打靶场吧。这两天抽时间把 vulhub 给搭建好。（这个 docker环境是把我整麻了。。。）然后在里面复现一些漏洞。

目标

和我简介里面写的一样，寒假还是想学到内网的部分，大概每天回来看一集小迪的课吧。感觉还是要多在靶场进行复现。看课只是最简单的一部分。有时间的话还想对python这个部分进行一些加强。感觉自己一直这个方面一直都学的不好。

下学期

下学期的话，再对寒假学习过的东西进行一些加强吧（top10 免杀 内网等），然后还是准备熬夜去刷刷src，如果可以找到CTF合适的队友的话，也可以去打打一些线上赛（感觉机会不大，而且我自身对CTF也不是很感兴趣😥😥😥）
然后就是就是搞一些蓝队的知识（溯源 应急响应之类的），主要是为hvv作一些准备。
还有像一些语言方面吧 听学长说安全还是要学一门主语言的 像 java go之类的，我现在对这些还不是太了解，等到那个程度再说吧。
对于下学期的规划的学习内容还是很多的，我也不知道可以完成多少。但是下学期应该上课比这学期更加摆烂了，大部分上课时间打算都用来搞安全了。感觉快大三了，和学校一些大佬相比，无论是知识的广度和深度，都还是不够。

为了防止我摆烂，打算把这篇规划置顶了，这样每次打开博客都可以想到今天的任务还没有完成。也欢迎大家监督我。💪💪💪💪
